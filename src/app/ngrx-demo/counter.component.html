NgRx Demo

This simplified counter uses a service backed by an RxJS BehaviorSubject to manage state. It illustrates the principles of unidirectional data flow: the component dispatches actions by calling methods on the store, and the view subscribes to the store’s observable state.

The following files in this project illustrate a typical NgRx setup:

  • counter.actions.ts – defines increment, decrement and reset actions.
  • counter.reducer.ts – a pure function that handles actions and returns new state.
  • counter.selectors.ts – derives the current count and doubled count.
