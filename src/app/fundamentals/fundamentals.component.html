Core Angular Fundamentals

  • Components & templates: Angular applications are built from components. A component combines a TypeScript class with an HTML template. Templates use interpolation, property binding, event binding and structural directives to display data and respond to user interaction.
  • Data binding: Angular supports one‑way data flow with interpolation and property binding, and two‑way data flow via [(ngModel)]. Event binding lets you respond to user actions.
  • Directives: Structural directives like *ngIf and *ngFor change the DOM by adding or removing elements. Attribute directives such as ngClass and ngStyle modify the appearance or behaviour of existing elements.
  • Pipes: Pipes transform values in templates. Angular includes many built‑in pipes for dates, numbers and internationalisation. You can also write custom pipes.
  • Services & dependency injection: Services provide reusable logic and data. Angular’s dependency injection system supplies services to components on demand.
  • Routing: The Angular router maps URLs to components and enables navigation across an application.
